复制对象

发表于 2021-03-01 更新于 2021-02-07 分类于 C++ ， Effective C++ ， 第二章：构造、析构、赋值运算
复制时要复制所有成员变量和 base class 成份
  • Copying 函数应该确保复制“对象内的所有成员变量”及“所有 base class 成份”
  • 不要尝试以某个 copying 函数实现另一个 copying 函数（特别是 copy 构造函数和 copy assignment 操作符）。应该将共同机能放进第三个函数中，并且由两个 copying 函数共同调用。

现在有一个类继承这个类，也要实现 copying 函数

可以看到，derived class 里面都调用了 base class 的 copying 函数，因为如果不调用，derived class 的 base class 部分没有被 copying ，将会自动调用无参数的构造函数，显然不满足 copying 的要求。
